Cara Main

Instruksi untuk Android

  1. 1

    Download aplikasi PowerTunnel

    Download PowerTunnel APK
  2. 2

    Klik tombol "Copy PowerTunnel" di atas untuk menyalin link host

  3. 3

    Buka PowerTunnel, tap icon 3 garis, lalu pilih Host Settings

  4. 4

    Paste link yang sudah di-copy, set Update Period ke "on start"

  5. 5

    Aktifkan VPN dan buka Growtopia untuk bermain! üéÆ

Instruksi untuk iOS

  1. 1

    Download aplikasi Surge 5 dari App Store

    Download Surge 5
  2. 2

    Klik tombol "Copy iOS Link" di atas untuk menyalin link konfigurasi

  3. 3

    Buka Surge 5, tap "Default.conf" lalu pilih "IMPORT" ‚Üí "Download Profile from URL"

  4. 4

    Paste link yang sudah di-copy dan ikuti setup VPN (Allow/OK)

  5. 5

    Aktifkan Surge dan buka Growtopia untuk bermain! üéÆ

Instruksi untuk Windows

  1. 1

    Klik tombol "Copy Windows Host" di atas untuk menyalin IP address

  2. 2

    Tekan Win + R, ketik notepad C:\Windows\System32\drivers\etc\hosts dan tekan Enter

    üí° Jika muncul error, buka Notepad as Administrator lalu open file hosts secara manual

  3. 3

    Scroll ke bawah file hosts dan paste IP address yang sudah di-copy di baris paling bawah

  4. 4

    Save file dengan Ctrl + S

  5. 5

    Buka Growtopia dan login untuk bermain! üéÆ
